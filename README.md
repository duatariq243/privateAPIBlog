## Running the Project

This project requires two servers to run: a backend API and a frontend server. You will need **two separate terminal windows**.

### 1. Start the Backend API
Navigate to the folder containing `index.js` and run:

```bash
nodemon index.js
This will start the backend API server on:
http://localhost:4000

2. Start the Frontend Server
Navigate to the folder containing server.js and run:

bash
Copy code
nodemon server.js
This will start the frontend server on:
http://localhost:3000
